# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능목록
(1) 랜덤텍스트 생성 기능

    - 자리수는 3자리로 한정한다.    
    - 0 ~ 9 까지의 자연수로 한정한다.
    - 중복되지 않아야 한다. 
    
(2) 스트라이크 여부 판정 기능

    - 스트라이크의 경우 1을 반환한다.
    - 스트라이크에 해당하지 않을 경우 0을 반환한다.
    
(3) 볼 여부 판정 기능

    - 볼의 경우 1을 반환한다.
    - 볼에 해당하지 않을 경우 0을 반환한다.
